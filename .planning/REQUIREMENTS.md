### Match Roundups

- [ ] **CONT-01**: Match summary with score and key events

- [ ] **CONT-03**: Top performer models for match
- [ ] **CONT-04**: LLM-generated narrative analysis
- [ ] **CONT-05**: Trigger on match completion
- [ ] **SEO-01**: Dynamic meta titles/descriptions
- [ ] **SEO-02**: Open Graph tags and images
- [ ] **SEO-03**: JSON-LD structured data (SportsEvent)
- [ ] **SEO-04**: ISR page generation with 60s revalidate

### Out of Scope

- [Trend analysis charts] — Performance over time (line charts), model comparison charts, streak detection
- [Advanced analytics] — Elo ratings, prediction confidence correlation, opponent-adjusted performance
- [Head-to-head comparisons] — Direct model vs model comparison
- [Anomaly detection] — Unusual prediction patterns, model degradation alerts
- [Interactive elements] — Expandable model details, related matches
- [Social sharing] — Twitter cards, shareable snippets
